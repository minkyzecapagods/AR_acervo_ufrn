Rede de Empréstimos de Livros

Análise interativa das relações entre livros baseada em padrões de empréstimo bibliotecário da UFRN

Sobre a Análise

Esta visualização explora relações entre livros baseadas em padrões sequenciais de empréstimo bibliotecário.

Metodologia: Livros conectados quando emprestados sequencialmente pelo mesmo usuário. Espessura das arestas indica frequência de transições.

Dados analisados:

  • 12,346 livros (nós)
  • 24,749 transições (arestas)
  • 3 categorias de usuários
  • Período: 2013-2020

Os dados revelam padrões de conhecimento interdisciplinar e caminhos de aprendizado preferenciais.
